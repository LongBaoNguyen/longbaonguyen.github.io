AP Computer Science Principles

This is a resource for teachers and students for AP Computer Science Principles. This is the current year 2018-2019, expanded version. To see last year 2017-2018 full lecture notes and resources, see this page, hosted on my faculty site at Umass Boston.

Course Outline

Hardware
An introduction to hardware including, BIOS, motherboards, peripherals, ports, CPU, registers, RAM, cache, hard disk drive(HDD), solid-state drive(SSD), number systems, encoding, and Unicode.
  1. Hardware (PPT, PDF)
  2. Binary and Encoding (PPT, PDF)
  3. Hardware II (PPT, PDF)
Programming
An introduction to programming using Processing(Java). A quick introduction to Processing can be found here.
  1. Introduction to Processing (PPT, PDF)
    • Drawing Basic Shapes
    • Basic Animation
  2. Basic Java Syntax (PPT, PDF)
    • Lab 1: Exact Change
    • Lab 2: Stopwatch
  3. Conditionals (PPT, PDF)
    • Keyboard Inputs
    • Mouse Inputs
  4. Methods (PPT, PDF)
  5. For Loops (PPT, PDF)
  6. While Loops and Math Functions (PPT, PDF)
  7. Arrays (PPT, PDF)
    • Lab 1: Many Balls
  8. Classes (PPT, PDF)
    • Object-Oriented with Processing
    • Lab 3: Multiple Classes Lab
  9. 2D Arrays (PPT, PDF)
    • Lab 2: 2D Grid
  10. Arraylists (PPT, PDF)
    • Lab 1: Disappearing Objects
    • Lab 2: Shoot Lab
  11. String and Recursion (PPT, PDF)
  12. Algorithms (PPT, PDF)
  13. Multimedia (PPT, PDF)

Introduction to Neural Networks

A 2-3 weeks curriculum designed for AP Computer Science Principles on Neural Networks. Students will implement from scratch a Python program that recognizes handwritten digits! The full course resources page can be found here.

Coding Challenges

A series of coding challenges. Most are mathematical in focus and require some trignometry. The distribution code is included and provides a template for completing the code.

  1. Ball Bounces Off Angles
    2. This animation challenge simulates a ball drop on an uneven ground. (Tutorial, Distribution Code)
  2. Inverse Kinematics: Pulling A String
    3. This animation challenge simulates pulling a string with the mouse. (Tutorial, Distribution Code)
